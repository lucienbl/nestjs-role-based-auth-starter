# NestJS Role Based Auth Starter

NestJS role based jwt auth starter! 🎉

## Features
- Role based JWT Auth
- MySQL database with TypeORM
- Swagger
- etc...

## Installation

```bash
$ npm install
```

## Running

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
